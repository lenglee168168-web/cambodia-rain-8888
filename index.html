✅ Step 3 — Enable Actions & Test

1. Go to your repo → click Actions tab
2. If it says "Actions disabled" → click Enable Actions
3. Click Cambodia Rain 8888 - Auto Weather Alert Bot
4. Click Run workflow → Run workflow (green button)
